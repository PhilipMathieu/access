# .cursor Documentation

This directory contains documentation for the Access project to help with development and understanding the codebase.

## Files

- **project.md**: Comprehensive project documentation including overview, structure, methodology, and setup
- **quick-reference.md**: Quick reference guide with common commands and key information

## Purpose

These documents are designed to help:
- Understand the project structure and purpose
- Set up the development environment
- Navigate the codebase and notebooks
- Understand the analysis workflow
- Reference key commands and file locations

## Project Summary

The Access project analyzes spatial accessibility to conservation land in Maine, calculating walking times from Census block centroids to conserved properties and analyzing demographic disparities, particularly for CEJST-identified disadvantaged communities.
